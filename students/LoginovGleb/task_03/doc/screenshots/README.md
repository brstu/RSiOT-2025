# Каталог скриншотов

В этом каталоге находятся все скриншоты, демонстрирующие выполнение лабораторной работы №3.

## Требуемые скриншоты (всего 16)

### Фаза 1: Развёртывание инфраструктуры и StatefulSet

1. `01_namespace_created.png` - Создан namespace
2. `02_pvc_bound.png` - PVC в состоянии Bound
3. `03_statefulset_ready.png` - StatefulSet готов
4. `04_pods_running.png` - Поды запущены

### Фаза 2: Проверка сохранности данных

1. `05_data_before_restart.png` - Данные до перезапуска пода
2. `06_pod_deletion.png` - Удаление пода
3. `07_pod_recreated.png` - Под пересоздан
4. `08_data_after_restart.png` - Данные после перезапуска (проверено сохранение)

### Фаза 3: Система резервного копирования

1. `09_backup_cronjob_created.png` - Создан CronJob для бэкапа
2. `10_backup_job_success.png` - Задача резервного копирования выполнена успешно
3. `11_backup_files_list.png` - Список файлов резервных копий

### Фаза 4: Система восстановления

1. `12_data_before_deletion.png` - Данные до удаления
2. `13_data_deleted.png` - Данные удалены (FLUSHALL)
3. `14_restore_job_running.png` - Задача восстановления выполняется
4. `15_restore_job_completed.png` - Задача восстановления завершена
5. `16_data_after_restore.png` - Данные после восстановления (проверено восстановление)

## Примечание

Скриншоты будут добавлены в процессе реального развёртывания и тестирования на кластере Kubernetes.
